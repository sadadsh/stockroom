# Stockroom

A KiCad V10 component-library manager for Windows. Drag in the zips you download
(symbol, footprint, 3D model), paste a product link, and Stockroom converts, names,
categorizes, links everything, fills every field, stores the datasheet and purchase
link, and registers it all with KiCad so the part is usable immediately. It also audits
an existing KiCad project, adding unrecognized parts and relinking recognized ones.

Design specification: [`docs/superpowers/specs/2026-07-12-stockroom-design.md`](docs/superpowers/specs/2026-07-12-stockroom-design.md).

Status: design approved, implementation not yet started.
